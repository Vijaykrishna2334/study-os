## 🌟 What is Study OS?

Study OS is a full-stack AI-powered learning platform built for engineers transitioning into AI/ML roles. It combines a **structured 12-phase curriculum** with an **autonomous job discovery and application system** powered by Google's Vertex AI Gemini 2.5 Flash.

### Two Products in One

| 📚 Study OS | 🚀 ApplyPilot |
|---|---|
| 340+ AI/ML topics across 12 phases | Real job scraping from LinkedIn, Indeed, Google Jobs |
| AI mock interviews & flashcards | AI scores every job against your skills (0-10) |
| DSA grinder for interview prep | AI rewrites your resume per job description |
| Chat with your own notes using AI | One-click apply + full application pipeline tracker |

## 🚀 Getting Started

### Prerequisites

- Node.js 20+
- Python 3.12+
- PostgreSQL 16+
- Google Cloud account with Vertex AI API enabled
- GCP Service Account JSON key file

### 1. Clone the repository

```bash
git clone https://github.com/Vijaykrishna2334/study-os.git
cd study-os
```

### 2. Install Node dependencies

```bash
npm install
```

### 3. Install Python job scraping dependencies

```bash
python3 -m venv /opt/jobspy-venv
/opt/jobspy-venv/bin/pip install --no-deps python-jobspy
/opt/jobspy-venv/bin/pip install pandas pydantic tls-client requests markdownify regex numpy
```

### 4. Configure environment variables

```bash
cp .env.example .env
```

Fill in your `.env`:

```env
# PostgreSQL
DATABASE_URL=postgresql://user:password@localhost:5432/studyos

# Google Cloud / Vertex AI
GCP_PROJECT=your-gcp-project-id
GCP_LOCATION=us-central1
GOOGLE_APPLICATION_CREDENTIALS=/path/to/gcp-key.json
LLM_PROVIDER=vertex

# Path to your markdown study notes
STUDY_ROOT=/path/to/your/notes

# Auth (NextAuth)
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-random-secret
```

### 5. Set up the database

```bash
npx prisma migrate deploy
npx prisma generate
```

### 6. Run

```bash
# Development
npm run dev

# Production
npm run build
cp -r .next/static .next/standalone/.next/static
cp -r public .next/standalone/public
cp -r scripts .next/standalone/scripts
pm2 start .next/standalone/server.js --name study-app
```

## 📖 How ApplyPilot Works

```
Step 1 — Upload Resume
  → PDF/DOCX/TXT parsed locally (pdf-parse + mammoth)
  → Saved to PostgreSQL

Step 2 — Discover Real Jobs
  → python-jobspy scrapes LinkedIn, Indeed, Google Jobs
  → 15 real jobs returned in ~20 seconds

Step 3 — AI Score Each Job
  → Gemini reads your mastered/weak topics from DB
  → Rates each job 0-10 for fit

Step 4 — AI Tailor Resume
  → Gemini rewrites your resume for that specific JD
  → Emphasizes relevant skills, never fabricates

Step 5 — Download
  → PDF / Markdown / plain text

Step 6 — Apply & Track
  → Click Apply → job URL opens
  → Mark Applied → saved to pipeline
  → 2-day reminder if no follow-up
```
